# Astro Recipe

> Your natal chart as a personal instruction manual.

An astrology app that gives users deep, actionable guidance based on their birth chart — built on real Western astrology algorithms, not generic horoscope copy.

**Problem:** Existing apps (Co-Star, The Pattern) are too vague. They describe who you are, not what to do.  
**Solution:** Natal chart reading that works like a professional astrologer — concrete, applied, niche-focused.  
**Audience:** Women 22–40, English-speaking, self-discovery oriented.  
**Platform:** Mobile-first web.  
**Status:** Pre-MVP — niche research in progress.
