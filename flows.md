## Flow 1 — Main Job
**"Отримати Glow Profile — зрозуміти свою природну мову через карту"**
Primary persona: Маша. Entry: перший запуск.

```mermaid
flowchart TD
    A([Старт — Welcome / Hook]) --> B[1.2 Введення дати народження]
    B --> C{Дата валідна?}
    C -- Ні --> B_err[error: невалідна дата]
    B_err --> B
    C -- Так --> D[1.3 Введення часу народження]
    D --> D_skip{Час відомий?}
    D_skip -- Не знаю --> D_approx[⚠️ Попередження: без часу результат буде частковим]
    D_approx --> E
    D_skip -- Так --> E[1.4 Введення місця народження]
    E --> E_load[loading: пошук міста...]
    E_load --> F{Місто знайдено?}
    F -- Ні --> E_err[error: місто не знайдено]
    E_err --> E
    F -- Так --> G[1.5 Питання: що зараз відбувається?]
    G --> G_skip{Відповів?}
    G_skip -- Пропустив --> H_load
    G_skip -- Відповів --> H_load[loading: розраховуємо Glow Profile...]
    H_load --> H{Карта розрахована?}
    H -- Помилка --> H_err[error: не вдалось розрахувати]
    H_err --> B
    H -- Успіх --> I[2.1 Wow-момент]
    I --> J[2.2 Твій стиль]
    J --> K[2.3 Твоя рутина]
    K --> L[2.4 Твоя енергія]
    L --> M[2.5 Твоя присутність]
    M --> N{Що далі?}
    N -- Глибше --> O[3.x Деталізація]
    N -- Поділитись --> P[4.1 Shareable Glow картка]
    N -- Готово --> Q([Glow Profile збережено ✅])
    O --> Q
    P --> Q
```

## Flow 2 — R1: Стиль і естетика
**"Знайти свою справжню візуальну мову"**
Entry: секція "Твій стиль" в Glow Profile.

```mermaid
flowchart TD
    A([Glow Profile]) --> B[2.2 Твій стиль]
    B --> B_check{Карта є?}
    B_check -- empty: карти нема --> onboard[Онбординг]
    onboard --> B
    B_check -- Так --> C[Wow: твій природній візуальний підпис]
    C --> D[Блок: твоя природна палітра і форми]
    D --> E[Блок: що тебе притягує естетично і чому]
    E --> F[Блок: конкретні інструкції — що носити, що уникати]
    F --> G{Хоче більше?}
    G -- Деталізація --> H[3.x Розділ карти — деталі]
    G -- Поділитись --> J[4.1 Shareable картка]
    G -- Назад --> A
    H --> A
    J --> A
```

## Нотатки

**Стани у кожному flow:**
- `loading` — розрахунок карти, пошук міста
- `error` — невалідна дата, місто не знайдено, помилка розрахунку
- `empty` — карта відсутня → redirect на онбординг
- `success` — Glow Profile отримано

**Тупики знайдено і виправлено:**
- Час невідомий → попередження + продовжуємо, не блокуємо
- Місто не знайдено → той самий крок, не скидаємо попередні дані
- Питання "що зараз?" пропущено → продовжуємо без відповіді
- Помилка розрахунку → повернення до введення дати з повідомленням
