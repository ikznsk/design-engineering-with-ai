## 1. Сутності продукту

| Сутність | Що це | Прив'язаний Job |
|----------|-------|-----------------|
| **Натальна карта** | Розрахована карта по даті/часу/місцю народження | Main Job — основа |
| **Glow Profile** | Персональний результат по ніші Glow | Main Job |
| **Стиль і естетика** | Яка твоя природна візуальна мова | R1 |
| **Рутина** | Які звички і ритм природні саме для тебе | R2 |
| **Енергія** | Що тебе відновлює і що виснажує | R3 |
| **Присутність** | Як ти виглядаєш для інших, яка твоя природна манера | R4 |
| **Wow-момент** | Перший екран результату — щось точне і несподіване | Emotional job + R1 |
| **Онбординг-питання** | "Що зараз відбувається в твоєму житті?" | Main Job (тригер) |
| **Shareable картка** | "Мій Glow тип" — для поширення | Social job |

**Під питанням (без чіткого Job → Backlog):**
- Синастрія / сумісність — є попит, але не в Glow-MVP
- Транзити / прогнози — recurring job не підтверджений для MVP

## 3. Навігаційна модель

### Глобальна навігація — 3 пункти

| Пункт | Що тут | Job |
|-------|--------|-----|
| **Glow** | Повний Glow Profile | Main Job |
| **Деталі** | Поглиблений розбір секцій карти | Main Job (глибина) |
| **Поділитись** | Shareable Glow картка | Social job |

Профіль — іконка в хедері, не в основній навігації.

### Підрахунок тапів — Main Job для Маші

```
Старт (Welcome)
  → тап 1: Введення дати
  → тап 2: Введення часу
  → тап 3: Введення місця
  → тап 4: Питання "що зараз?"
  → тап 5: Glow Profile ✅
```

Онбординг: 5 кроків — більше 3, але виправдано: без дати/часу/місця карта неможлива.
Після першого разу: Main Job у 1 тап (Glow в навігації → одразу результат).
