# Astro Recipe — Product Brief

## Concept
An astrology app that turns a user's natal chart into a personal instruction manual for life.
The user enters their birth date + time → receives a deep, structured analysis built on Western astrology algorithms → gets concrete, actionable guidance across life areas.

## Problem
Apps like Co-Star and The Pattern are too abstract and generic:
- Readings are vague and apply to everyone
- They describe WHO you are but not WHAT TO DO
- No practical application — you read it and move on

## Solution
Natal chart as a personal operating system. Algorithmic interpretation that reads planetary positions, aspects, houses, and signs in combination — the way a professional astrologer would — and turns it into specific, applied guidance for real questions people actually have.

## Core User Flow
1. User enters birth date + time + place
2. App calculates natal chart
3. User receives deep structured analysis
4. Analysis organized into life areas with concrete instructions

## Audience
- Women, 22–40, English-speaking (MVP)
- Self-discovery oriented
- Frustrated with vague/pop astrology and fake astrologers
- Looking for truth, genuine self-knowledge, and practical application

## Platform
Mobile-first web → responsive desktop

## Differentiation
- Concrete instructions, not abstract descriptions
- Reads planets, aspects, houses in combination (not in isolation)
- Answers real questions people bring to astrologers
- Niche focus — deep expertise in one area rather than shallow coverage of everything
